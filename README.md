# DuploVim
DuploVim is a very easy and customizable neovim configuration

# Requirements
## Lsps (optional)
- ccls
- lua-language-server
- pylyzer

- Nerd Font (very important, but optional)
- python3 provider in neovim
- neovim (latest)

# Installation
- run ```git clone https://github.com/DuploVim/DuploVim ~/.config/nvim```
- run ```nvim```
- Enjoy! =)
